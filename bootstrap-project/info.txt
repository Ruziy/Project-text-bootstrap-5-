.container - создает АДАПТИВНЫЙ блок с отступами
.row - display-flex + jc:sb
.col - весь контент поделён на 12 столбец, с помощью col-12 вы определяем, что блок будет занимать /n
всю ширину 
#footer(пример) - якорь который подключается через href,применяет плавную прокрутку до блока с таким якорем
popap = modal window

Всплывающие меню = аккордеон
Bootstrap=>components=>first
Чтобы сделать элемент открытым убираем класс collapsed из accordion-button, 
добавляем класс show в accordion-collapse
Важно!! если несколько аккордеонов, чтобы они не перекликались между собой
1.Нужно изменять наследуемый id у главного блока class="accordion accordion-flush",
на одинаковые значения с элементами аккордеона в data-bs-parent
2.Также одинаковые значения должны иметь data-bs-target="#...",aria-controls="....",
id="...." в блоке class="accordion-collapse collapse show"
